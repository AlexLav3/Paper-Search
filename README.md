# Paper-tracker, PubMed Research Paper Search Tool

A JavaScript tool to search and fetch academic papers from Europe PMC API.
Currently in development

## What It Does

- Searches biomedical literature from Europe PMC database
- Fetches multiple pages of results automatically
- Extracts key information: title, authors, abstract, journal, year

## Requirements

- Node.js v12 or higher
- `node-fetch` package (if using Node < 18)
